Mogul Typeface


Exploration of letterforms and type // Type Design

Intructor:  James Montablo

This project was for ‘Typeface Design’(2019). This class focused on constructing and designing of a Latin Typeface using the program FontLab VI for the duration of the semester.

Mogul was designed to be both a text and display typeface that  is enjoyable to read. It features wide lowercases, high contrast  in each glyph, thin and elegant serifs and rounder forms. To showcase this typeface, the specimen features extracts from the thirteenth chapter of The Little Prince by Antoine de Saint-Exupéry in four  different languages.
