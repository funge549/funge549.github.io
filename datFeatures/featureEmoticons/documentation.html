1.2 adding jquery

_ in the same function add the following lines to activate the divs and add the dat.emoticon into your text box.
_ this will also make the selected button be 'marked' as active.
_ we are adding the emoticons to the textarea with the id 'txtarea' make sure you add this id! 

$(".button").click(function() {
		var emoticon = $(this)
		emoticon.siblings().removeClass('selected')
		emoticon.addClass('selected')

		var text = emoticon[0].innerText;

		$('#txtarea').val($('#txtarea').val() + " " + text)

		console.log(text)
	})
})

(*) you will have to add the latest jquery doc to your index.html find and learn more about this here
